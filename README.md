# es_admin2 onesync only
A updated and better version of es_admin2

## Download & Installation

### Using [fvm](https://github.com/qlaffont/fvm-installer)
```
fvm install --save --folder=Admin known-001/es_admin2
```

### Using Git
```
cd resources
git clone https://github.com/known-001/es_admin2 [Admin]/es_admin2
```

### Manually
- Download https://github.com/known-001/es_admin2
- Put it in the `[Admin]` directory


## Installation
- Add this to your server.cfg:

```
start es_admin2
```
